Blocks are reusable pieces of formatted content—organization bios, project descriptions, budget tables, legal details—that you write once and insert into any proposal. Rather than retyping the same information across multiple applications, you keep a library of blocks and drop them in wherever they're needed.

## Creating a block

Click **+ New Block** and choose one of two methods.

<AccordionGroup>
  <Accordion title="Upload Block">
    Upload an existing document and Allegory will parse it into an editable block, preserving structure and formatting.

    Supported upload formats:

    | Format | Extensions |
    |--------|------------|
    | PDF | `.pdf` |
    | Word | `.doc`, `.docx`, `.docm` |
    | Rich Text | `.rtf` |
    | Plain text | `.txt` |
    | HTML | `.html` |

    <Note>
      Parsing can take up to 5 minutes for longer documents.
    </Note>
  </Accordion>
  <Accordion title="Start from Scratch">
    Opens a blank rich text editor where you build the block from the ground up. The block auto-saves as you work.
  </Accordion>
</AccordionGroup>

## Editing a block

The block editor has a full formatting toolbar. Click any block in your library to open it.

**Text formatting**

Bold, italic, underline, strikethrough, text color, and highlight are available from the toolbar. Use the **Turn into** dropdown to change a paragraph into a heading (H1–H3), a bulleted list, a numbered list, or a blockquote.

**Tables**

Click the table icon in the toolbar to insert a table. Once a table is selected, right-click or use the **Table Options** menu to:

- Add or delete rows and columns
- Merge or split cells
- Delete the table

**Undo / Redo**

Use **⌘Z** / **Ctrl+Z** to undo and **⌘Shift+Z** / **Ctrl+Y** to redo. The toolbar also shows undo and redo buttons.

## Using blocks in proposals

You can insert a block anywhere in a proposal from the Proposal tab.

**From the floating toolbar:** Highlight any text (or click to place your cursor), then click **Use Block** in the floating toolbar that appears.

**From the static toolbar:** Click the **⋯** overflow menu at the end of the toolbar and select **Use Block**.

In both cases, a dialog appears listing your saved blocks. Select the one you want and click **Use Block** to insert it at your cursor position, with its original formatting intact.

<Tip>
  Use AI Context in Documents for dynamic, grant-specific content—mission statements, impact data—and use Blocks for static, reusable sections like your organization's legal name, address, or standard budget format.
</Tip>
