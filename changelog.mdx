<Update label="April 2026" description="Scout multi-session support" tags={["New Releases"]}>
  ## Scout multi-session support

  You can now run multiple Scout conversations in parallel. Start a new session at any time from the sessions panel, give it a custom name, and switch between sessions without losing context. Each session maintains its own history, mode, and research state — so you can keep a long-running search going while starting a fresh conversation for a different project.

  Learn more about [Scout sessions](/scout/overview).
</Update>

<Update label="April 2026" description="PARP grants in Scout Search" tags={["New Releases"]}>
  ## PARP grants in Scout Search

  Scout Search now includes the Polish Agency for Enterprise Development (PARP) as a data source. When you search for funding opportunities, Scout can surface EU-funded programmes administered by PARP — including FENG, FEPW, FERS, and Norwegian Funds — alongside results from Grants.gov, SAM.gov, SBIR, EU Funding & Tenders, and the Allegory funder database. Results from PARP are automatically translated from Polish to English.

  See the full list of [sources Scout searches](/scout/search).
</Update>

<Update label="April 2026" description="Scout 1.0" tags={["New Releases"]}>
  ## Scout 1.0

  Scout is Allegory's native AI agent, built to power your everyday grant workflows. Scout lives in your sidebar and operates across three specialised modes — [Ask](/scout/ask) for quick grant questions, [Search](/scout/search) for deep multi-source opportunity discovery, and [Writing](/scout/writing) for drafting and refining proposals grounded in your organisation's own documents.

  Unlike a standard chatbot, Scout maintains state across conversations. It remembers what it has researched, which opportunities it has found, and what it is currently working on. When it discovers a new opportunity, it asks for your approval before saving it to your Funding Tracker. When it writes, it cites every source.

  Alongside Scout 1.0, we've also launched a new home page in the app that puts Scout front and centre — making it the fastest way to get from zero to a funded opportunity.

  Read the [Scout overview](/scout/overview) to get started.
</Update>

<Update label="March 2026" description="Funding tracker endpoints" tags={["New Releases"]}>
  ## Funding tracker endpoints

  You can now create, manage, and look up opportunities in your Allegory [Funding Tracker](/funding-tracker/funding-opportunities) through the API. New endpoints include:

  - `POST /v1/opportunities` to create a new opportunity
  - `GET /v1/opportunities` to list and filter opportunities
  - `PUT /v1/opportunities/:id` to update an existing opportunity
  - `GET /v1/opportunities/lookup` to look up an opportunity by external reference

  See the [Funding Opportunities API reference](/api-reference/opportunities/list-opportunities) for details.
</Update>

<Update label="February 2026" description="Initial API release" tags={["New Releases"]}>
  ## Allegory API launch

  The Allegory Funder API is live. Search private and public funders, retrieve detailed funder profiles by EIN, and access grant history data from IRS 990 filings.

  - `GET /v1/funders` to search funders by name, type, and location
  - `GET /v1/funders/:ein` to get a funder by EIN
  - `GET /v1/funders/:ein/history` to get grant history for a specific funder

  Generate your API key from [Account → Security & Access](/account/security-access) and explore the [API reference](/api-reference/introduction).
</Update>
