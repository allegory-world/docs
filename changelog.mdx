<Update label="March 2026" description="Funding tracker endpoints" tags={["New Releases"]}>
  ## Funding tracker endpoints

  You can now create, manage, and look up opportunities in your Allegory Funding Tracker through the API. New endpoints include:

  - `POST /v1/opportunities` to create a new opportunity
  - `GET /v1/opportunities` to list and filter opportunities
  - `PUT /v1/opportunities/:id` to update an existing opportunity
  - `GET /v1/opportunities/lookup` to look up an opportunity by external reference

  See the [Funding Opportunities API reference](/api-reference/opportunities/list-opportunities) for details.
</Update>

<Update label="February 2026" description="Funder profiles" tags={["New Releases"]}>
  ## Funder profiles in Allegory

  We launched funder profiles, giving you a dedicated place to review and work with funder information across three areas:

  - **Profile** — Key organization information in one place
  - **People** — A new People view with improved contact enrichment to help you identify the right contacts behind each organization
  - **History** — Funding history with support for manual additions and clickable recipient EINs, making it easier to verify records and work with historical grant data

<Update label="February 2026" description="Initial API release" tags={["New Releases"]}>
  ## Allegory API launch

  The Allegory Funder API is live. Search private and public funders, retrieve detailed funder profiles by EIN, and access grant history data from IRS 990 filings.

  - `GET /v1/funders` to search funders by name, type, and location
  - `GET /v1/funders/:ein` to get a funder by EIN
  - `GET /v1/funders/:ein/history` to get grant history for a specific funder

  Get started with the [Quickstart guide](/quickstart) or explore the [API reference](/api-reference/introduction).
</Update>
