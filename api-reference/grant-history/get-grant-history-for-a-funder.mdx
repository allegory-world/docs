---
openapi: GET /v1/funders/{ein}/history
---

Retrieve the grant-making history for a specific funder. This endpoint provides access to historical grant data showing who the funder has supported and how much they've given.

## Data sources

Grant history data is sourced from regulatory filings (990 and 990-PF), live web search, and public and private APIs. The data includes:
- Grant recipients and their EINs
- Grant amounts (cash and non-cash)
- Grant purposes and descriptions
- Transaction years

## Examples

<RequestExample>
```bash cURL
curl -X GET 'https://api.allegory.world/v1/funders/131635294/history?limit=100&offset=0' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -H 'Accept: application/json'
```
</RequestExample>

## Pagination

This endpoint supports pagination with a maximum of 1000 records per page:

- **Default limit**: 100 grants per page
- **Maximum limit**: 1000 grants per page
- Use `offset` to navigate through pages (e.g., page 2: `offset=100`)

<Tip>
For funders with extensive grant history, use the `limit` parameter to fetch larger pages (up to 1000) to reduce the number of API calls needed.
</Tip>
