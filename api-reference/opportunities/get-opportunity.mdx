---
openapi: GET /v1/opportunities/{id}
---

Retrieve detailed information about a specific funding opportunity by its UUID.

## Use cases

- Get full details of a funding opportunity
- Retrieve an opportunity for editing
- Check current status and deadlines
- Access all metadata for an opportunity

## Examples

```bash cURL
  -H 'X-API-Key: YOUR_API_KEY' \
  -H 'Accept: application/json'
```
</RequestExample>

## Path parameter

The `id` parameter must be a valid UUID for an opportunity that belongs to your organization.

<Warning>
You can only access opportunities that belong to your organization. Attempting to access another organization's opportunities will return a 404 error.
</Warning>
