---
openapi: GET /v1/opportunities
---

List funding opportunities for your organization with powerful filtering options. This endpoint provides access to your organization-scoped funding opportunities.

## Use cases

- Browse all funding opportunities for your organization
- Filter by donor, program name, or status
- Search by funding type (public, private, or other)
- Filter by country to find local opportunities
- Paginate through large result sets

## Examples

<RequestExample>
```bash cURL
curl -X GET 'https://api.allegory.world/v1/opportunities?funding_type=private&limit=20' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -H 'Accept: application/json'
```
</RequestExample>

## Filtering options

Combine multiple filters to narrow your search:

- **Donor**: Partial match on donor name (e.g., `donor=Foundation`)
- **Program Name**: Partial match on program name (e.g., `program_name=Education`)
- **Status**: Filter by opportunity status (e.g., `status=active`)
- **Funding Type**: Filter by type - `public`, `private`, or `other`
- **Country**: Partial match on country (e.g., `country=United States`)

<Tip>
Use the `limit` parameter to control page size (1-100, default: 20) and `offset` for pagination.
</Tip>
