---
openapi: GET /v1/funders/{ein}
---

Retrieve detailed information about a nonprofit funder by their Employer Identification Number (EIN). This endpoint provides direct lookup for a specific funder.

## EIN format

The EIN (Employer Identification Number) is a 9-digit number assigned by the IRS. You can find EINs on:
- IRS Form 990 filings
- Charity Navigator profiles
- Organization websites (often in their footer or About page)

<Warning>
EINs must be provided as strings, not numbers, to preserve leading zeros. For example, use `"012345678"` not `12345678`.
</Warning>

## Examples

<RequestExample>
```bash cURL
curl -X GET 'https://allegory-middleware.workers.dev/v1/funders/131635294' \
  -H 'Accept: application/json'
```
</RequestExample>

</ResponseExample>

<Tip>
After retrieving a funder by EIN, you can use the same EIN to fetch their grant history using the `/v1/funders/{ein}/history` endpoint.
</Tip>
